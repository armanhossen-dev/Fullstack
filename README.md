## 💻 Get Mac's Local IP Address:
Open ** terminal.app ** and run:

```bash
ipconfig getifaddr en0
```
## OR try this bro
```bash
ipconfig getifaddr en1
```

## 💻 Get Window's Local IP Address:

Open **Command Prompt /' CMD ** and run:
```bash
ipconfig
```
